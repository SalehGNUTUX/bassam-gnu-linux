# Run the application in development mode

**Requirements**

- nodejs

- git

```bash
git clone https://github.com/SalehGNUTUX/bassam-gnu-linux
cd bassam-gnu-linux
npm i
npm run dev
```

<br>

 **build**

<br>

```bash
#Windows
npm run packwin
#Linux
npm run packlinux
```
